**Onyx Launcher** to lekki, nowoczesny i zorientowany na wydajność manager Minecrafta stworzony w Pythonie. Został zaprojektowany, aby uprościć zarządzanie modami i wersjami gry, zachowując przy tym minimalistyczny, ciemny design.

---

## 🚀 Kluczowe Funkcje

* **⚡ Szybki Start:** Błyskawiczne uruchamianie dowolnej wersji Minecrafta (Vanilla, Fabric, Forge).
* **📦 Mod Manager (Modrinth API):** Wyszukiwarka i instalator modów bezpośrednio w aplikacji.
* **🧩 Inteligentne Aktualizacje:** Przycisk "ZAKTUALIZUJ", który automatycznie podmienia pliki modów na nowsze wersje.
* **🧪 Auto Fabric-API:** Automatyczne pobieranie wymaganego Fabric API przy instalacji modów Fabric.
* **📊 Manager RAM:** Dynamiczne przypisywanie pamięci RAM dla gry za pomocą suwaka.
* **📂 Izolowane Środowisko:** Wszystkie pliki (logs, configs, screenshots) są bezpiecznie przechowywane w folderze `.minecraft`.
